Introduction

  • HTML stands for Hyper Text Markup Language
  • HTML describes the structure of the webpage
  • HTML uses the elements to tell brower to display the content.These elements are called as"tags"
  • HTML tags acts as label for the content

HTML tag

The HTML tag looks like <'tagname'> content <'/tagname'>.
Where <'tagname'> stands for starting tag and <'/tagname'> stands for ending tag.The content shoud dispaly in between starting and ending tags

Description of the tags

  • the <'!DOCTYPE html'> tells the brower that the document using is HTML5
  • the <'html'> is the root tag for html document
  • the <'head'> consist of the information about the HTML document
  • the <'title'> gives the title for the HTML page
  • the <'body'> consist of the visible part of the HTML document
  • the <'h1'> used to specify the heading
  • the <'p'> used to specify the paragraph
